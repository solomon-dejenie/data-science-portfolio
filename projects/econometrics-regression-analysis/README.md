# 📉 Econometrics & Regression Analysis

## 🎯 Project Overview
This project applies econometric and statistical modeling techniques to analyze relationships between economic variables using regression analysis.

---

## 🧠 Objectives
- Build multiple linear regression models
- Evaluate statistical significance of variables
- Interpret economic relationships
- Validate model performance

---

## 🛠️ Tools & Technologies
- Python
- Statsmodels
- Pandas
- NumPy
- Statistical inference techniques

---

## 📊 Key Findings
- Identified key predictors influencing the dependent variable
- Evaluated model accuracy using R² and p-values
- Improved model interpretability through statistical diagnostics

---

## 💼 Business Value
Supports economic forecasting and decision-making by quantifying relationships between key variables.
