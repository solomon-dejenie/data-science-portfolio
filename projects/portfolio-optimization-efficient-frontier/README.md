# 📊 Portfolio Optimization & Efficient Frontier

## 🎯 Project Overview
This project implements Modern Portfolio Theory (MPT) using Python to construct an optimized investment portfolio. It focuses on balancing risk and return by simulating multiple asset combinations and identifying the most efficient portfolio using the Sharpe Ratio.

---

## 🧠 Objectives
- Analyze historical asset returns
- Compute portfolio risk and return
- Simulate multiple portfolio combinations
- Identify the Efficient Frontier
- Determine optimal portfolio allocation

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Financial modeling techniques

---

## 📈 Key Outcome
- Built a data-driven portfolio optimization model
- Visualized Efficient Frontier
- Identified optimal risk-adjusted portfolio

---

## 💼 Business Value
This project demonstrates how quantitative methods can be used to support investment decision-making and risk management strategies in financial markets.
